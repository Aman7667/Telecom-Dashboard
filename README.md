# Telecom-Dashboard
📡 TELECOME DATA Dashboard
📌 Project Overview

The TELECOME DATA Dashboard is an interactive Power BI dashboard developed to analyze telecom customer behavior, usage patterns, billing performance, and customer satisfaction metrics. The dashboard transforms telecom data into meaningful insights using dynamic KPIs, charts, and interactive filters.

This dashboard helps telecom companies monitor customer engagement, understand usage trends, optimize subscription plans, and improve decision-making through visual analytics.

🎯 Objective

The primary objective of this dashboard is to:

Monitor customer activity and telecom usage trends
Analyze call usage and data consumption
Evaluate customer satisfaction levels
Compare plan types and demographic distribution
Track monthly billing performance
Support strategic business decisions using data insights
🛠️ Tools & Technologies Used
📊 Power BI Desktop – Used for creating interactive dashboards and visual reports.
📂 Power Query – Applied for data cleaning and transformation.
🧠 DAX (Data Analysis Expressions) – Used for calculated KPIs and measures.
🔗 Data Modeling – Relationships created between telecom datasets for analysis.
📁 PBIX File – Used as the dashboard development file format.
📂 Dataset Information

The telecom dataset contains information related to:

Customer IDs and customer names
Customer satisfaction scores
Call usage minutes
Monthly billing amount
Data usage in GB
Plan duration
Subscription plan types
Customer age and gender
City-wise customer distribution

The data is structured to support multi-dimensional telecom analytics.

🔍 Dashboard Features & Visual Analysis
🔹 Customer Satisfaction Filter

An interactive satisfaction score slider allows users to filter customer insights between scores ranging from 1 to 10.

This helps telecom providers analyze customer behavior based on satisfaction levels.

🔹 Monthly Call Usage Analysis

The bar chart visualizes the maximum age and maximum call usage minutes across different months.

Key Insights:
Call usage remains consistently high throughout the year.
Peak usage reaches approximately 1000 minutes during several months.
Monthly customer age trends remain stable.
Business Impact:

This analysis helps telecom providers understand seasonal demand and network usage patterns.

Gender
Plan Type
